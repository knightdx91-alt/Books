# Scale and Silver — earlier/fuller draft (deleted-scenes reference)

**What this is:** a snapshot of the Book One manuscript (29 chapters, ~151,950 words) from the
`scale-and-silver` branch of the old `The-Saeren-Chronicles` repo — an **earlier draft** taken
before the canonical version was trimmed/tightened.

**Why it's kept:** it is NOT canon. The canonical manuscript lives in
`books/scale-and-silver/manuscript/chapters/`. This earlier draft is preserved purely as a
**reference for cut prose ("deleted scenes")** — several early chapters here contain passages
that were edited out of the canonical version and exist nowhere else.

**Relationship to canon (verified 2026-07-13):**
- Early chapters (1, 3–6, 13–17, …): **this draft is fuller** — canon trimmed these (e.g., in
  ch.5, a reflective "She sat by the water and let herself understand what she had done…" passage
  that canon cut).
- Later chapters (19, 21, 28, 29): **canon is fuller** — those were expanded after this snapshot.
- Total: canon ~153,460 words vs this draft ~151,950; canon also has the full publishing package
  (editorial, covers, print PDFs, epub) that this draft lacks.

Saved here so the source branch can be deleted without losing the cut prose. Do not treat as
canon; mine it if a trimmed passage is ever wanted back.
